# Conceptual Framework: Wildfire Risk Modeling (Kanchanaburi, Thailand)
## 1.	คำถามวิจัย (Research Question)
งานวิจัยนี้มีวัตถุประสงค์เพื่อตอบคำถามว่า: 
“พื้นที่ใดในจังหวัดกาญจนบุรีมีความเสี่ยงต่อการเกิดไฟป่าสูงในช่วงเดือนมกราคม - พฤษภาคม ปี 2024”
ตอบได้ด้วย แบบจำลองเชิงพื้นที่ (Spatial Model) ซึ่งจะแสดงผลในรูปแบบแผนที่ความเสี่ยง
## 2.	ปัจจัยที่มีผลต่อการเกิดไฟป่า (Factors)
แบบจำลองนี้เลือกใช้ปัจจัยทั้งหมด 4 ตัว โดยอิงจากหลักการทางวิทยาศาสตร์ของการเกิดไฟป่า

2.1.	ความแห้งของเชื้อเพลิง (Dryness: NBR) : คำนวณจากดัชนี Normalized Burn Ratio (NBR) จากข้อมูล Landsat ใช้เป็นตัวแทนของ ความชื้นในเชื้อเพลิง โดยพื้นที่ที่มีค่าความแห้งสูงจะมีโอกาสเกิดไฟป่าสูง

2.2.	อุณหภูมิพื้นผิว (Land Surface Temperature: LST) : ใช้ข้อมูลจาก MODIS อุณหภูมิที่สูงส่งผลให้เชื้อเพลิงสูญเสียความชื้นและเพิ่มโอกาสในการติดไฟ

2.3.	ปริมาณเชื้อเพลิง (Fuel: NDVI) : คำนวณจาก NDVI จากข้อมูล Landsat ใช้เป็นตัวแทนของปริมาณพืชพรรณ (ชีวมวล) ซึ่งเป็นแหล่งเชื้อเพลิงในการเกิดไฟป่า

2.4.	ประวัติการเกิดไฟป่า (Fire History) : ใช้ข้อมูล hotspot จาก FIRMS (ปี 2022 - 2023) สะท้อนถึงพื้นที่ที่มีแนวโน้มเกิดไฟซ้ำในอดีต

## 3.	การกำหนดน้ำหนัก (Weighting Scheme)
โมเดลใช้วิธี Weighted Linear Combination (WLC) โดยกำหนดน้ำหนักดังนี้:
-	Dryness (NBR) = 0.30
-	LST = 0.25
-	NDVI = 0.25
-	Fire History = 0.20
  
เหตุผลในการกำหนดน้ำหนัก: 
-	ความแห้งของเชื้อเพลิงเป็นปัจจัยหลักที่ควบคุมการติดไฟ จึงได้รับน้ำหนักสูงที่สุด
-	อุณหภูมิและปริมาณเชื้อเพลิงเป็นปัจจัยสนับสนุนที่สำคัญ
-	ส่วนประวัติไฟป่าใช้เพื่อสะท้อนแนวโน้มเชิงพื้นที่ในอดีต
## 4.	แนวทางการตรวจสอบโมเดล (Validation Strategy)
โมเดลจะถูกตรวจสอบโดยใช้ข้อมูล MODIS Burned Area (MCD64A1) ซึ่งแสดงพื้นที่ที่เกิดไฟจริง

วิธีการตรวจสอบ : เปรียบเทียบพื้นที่ที่โมเดลทำนายว่า “มีความเสี่ยงสูง” กับพื้นที่ที่เกิดไฟจริง วิเคราะห์ความสอดคล้องของทั้งสองข้อมูล

เหตุผลในการเลือกวิธีนี้ : เป็นวิธีที่สามารถใช้ข้อมูลจริงมาตรวจสอบความถูกต้องของโมเดลได้โดยตรง และเหมาะสมกับการวิเคราะห์เชิงพื้นที่
## 5.	สรุปกรอบแนวคิด
โมเดลนี้พัฒนาขึ้นโดยใช้ปัจจัยด้านความแห้ง อุณหภูมิ ปริมาณเชื้อเพลิง และประวัติไฟป่า ซึ่งเป็นองค์ประกอบหลักของการเกิดไฟป่า โดยใช้วิธี Weighted Linear Combination เพื่อรวมปัจจัยทั้งหมดเข้าด้วยกัน และสามารถสร้างแผนที่ความเสี่ยงเพื่อใช้ในการวิเคราะห์และวางแผนจัดการไฟป่าได้
