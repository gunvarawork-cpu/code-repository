# Wildfire Risk Modeling (Kanchanaburi, Thailand)
# ภารกิจที่ 1: กำหนดกรอบแนวคิด (Conceptual Framework)
## 1.1	คำถามวิจัย 
งานวิจัยนี้มีวัตถุประสงค์เพื่อตอบคำถามว่า:

#### พื้นที่ใดในจังหวัดกาญจนบุรีมีความเสี่ยงต่อการเกิดไฟป่าสูงในช่วงเดือนมกราคม - พฤษภาคม ปี 2024

สามารถตอบได้ด้วย แบบจำลองเชิงพื้นที่ (Spatial Model) ซึ่งแสดงผลในรูปแบบแผนที่ความเสี่ยง และแบ่งระดับความเสี่ยงออกเป็น 3 ระดับ ได้แก่ ต่ำ ปานกลาง และสูง
## 1.2	ปัจจัยที่มีผลต่อการเกิดไฟป่า
แบบจำลองนี้ใช้ปัจจัยทั้งหมด 4 ตัว โดยอิงจากข้อมูลดาวเทียมและหลักการทางวิทยาศาสตร์ ดังนี้:
1)	ความแห้งของเชื้อเพลิง (Dryness: NBR) : คำนวณจากดัชนี Normalized Burn Ratio (NBR) จากข้อมูล Landsat มีการ normalize และกลับค่า (invert) เพื่อให้ ค่าที่สูง = ความแห้งสูง = ความเสี่ยงสูง ใช้เป็นตัวแทนของความชื้นในเชื้อเพลิง ซึ่งเป็นปัจจัยสำคัญต่อการเกิดไฟป่
2)	อุณหภูมิพื้นผิว (Land Surface Temperature: LST) : ใช้ข้อมูลจาก MODIS อุณหภูมิที่สูงขึ้นส่งผลให้เชื้อเพลิงสูญเสียความชื้นและติดไฟได้ง่ายขึ้น
3)	ปริมาณเชื้อเพลิง (Fuel: NDVI) : คำนวณจากดัชนี NDVI จาก Landsat ใช้เป็นตัวแทน (proxy) ของปริมาณพืชพรรณซึ่งเป็นเชื้อเพลิง ค่า NDVI สูง หมายถึงมีชีวมวลมาก และเพิ่มโอกาสการเกิดไฟเมื่อสภาพแวดล้อมแห้ง
4)	ประวัติการเกิดไฟป่า (Fire History) : ใช้จำนวน hotspot สะสม (count) จาก FIRMS ในช่วงปี 2022 - 2023 ใช้เป็นตัวแทนของพื้นที่ที่มีแนวโน้มเกิดไฟซ้ำในอดีต
## 1.3	การกำหนดน้ำหนัก (Weighting)
โมเดลใช้วิธี Weighted Linear Combination (WLC) โดยกำหนดน้ำหนักดังนี้:
-	Dryness = 0.30
-	LST = 0.25
-	NDVI = 0.25
-	Fire History = 0.20
เหตุผล: ความแห้งของเชื้อเพลิงเป็นปัจจัยหลักที่ควบคุมการติดไฟ ขณะที่อุณหภูมิและปริมาณเชื้อเพลิงเป็นปัจจัยสนับสนุน ส่วนประวัติไฟป่าใช้เพื่อสะท้อนแนวโน้มเชิงพื้นที่ในอดีต
## 1.4	การตรวจสอบความถูกต้องของโมเดล (Validation)
ใช้ข้อมูล MODIS Burned Area (MCD64A1, ปี 2024) เป็นข้อมูลอ้างอิง
วิธีการ:
-	สร้างแผนที่พื้นที่ที่เกิดไฟจริง (burned area)
-	เปรียบเทียบกับพื้นที่ที่โมเดลทำนายว่า “เสี่ยงสูง” (risk ≥ 0.3)
-	คำนวณจำนวนพิกเซลที่ทำนายถูก (intersection)
ตัวชี้วัด
Accuracy = พื้นที่ที่ทำนายถูก/พื้นที่ที่เกิดไฟทั้งหมด × 100
ผลลัพธ์:
-	Total burned pixels ≈ 4600
-	Correct predicted ≈ 3148
-	Accuracy ≈ 68.45%
# ภารกิจที่ 2: การสร้าง Spatial Model
## 2.1 การ Normalize
ข้อมูลทุกปัจจัยถูกปรับให้อยู่ในช่วง 0 - 1 เพื่อให้สามารถนำมารวมกันได้:
-	LST: 13000 - 16000
-	NDVI: 0 - 0.8
-	NBR: scale + invert เพื่อแทนความแห้ง
-	Fire history: 0 - 5
## 2.2 การรวมปัจจัย (Model Equation)
Risk= (0.30 × Dryness) + (0.25 × LST) + (0.25 × NDVI) + (0.20 × History)
## 2.3 การแบ่งระดับความเสี่ยง
-	Low: < 0.3
-	Medium: 0.3 – 0.5
-	High: > 0.5
ช่วยให้สามารถตีความผลลัพธ์ได้ง่ายและนำไปใช้งานได้จริง
## 2.4 ผลลัพธ์
ได้แผนที่ 2 รูปแบบ:
### •	Wildfire Risk
o	สีน้ำเงิน = Low Risk

o	สีเหลือง = Moderate Risk

o	สีแดง = High Risk
### •	Risk Classification (3 ระดับ)
o	สีเขียว = Low (< 0.3)

o	สีส้ม = Medium (0.3 – 0.5)

o	สีแดง = High (> 0.5)
# ภารกิจที่ 3: Sensitivity Analysis

วิธีการ:
-	ปรับน้ำหนักของ Dryness ±20%
-	คำนวณความแตกต่างของค่าความเสี่ยง (ΔRisk)

เกณฑ์:
-	Δ < 0.03 → Robust สีเขียว
-	Δ ≥ 0.03 → Sensitive สีแดง

ผลลัพธ์: Model Sensitivity
-	พื้นที่ส่วนใหญ่เป็น Robust (เสถียร)
-	มีบางพื้นที่เป็น Sensitive (ไม่แน่นอน)

การตีความ: โมเดลมีความเสถียรในภาพรวม แต่ยังมีบางพื้นที่ที่ไวต่อการเปลี่ยนน้ำหนัก แสดงให้เห็นว่า Dryness เป็นปัจจัยที่มีอิทธิพลสูงต่อผลลัพธ์
# ภารกิจที่ 4: Validation กับข้อมูลจริง
ผลลัพธ์ : Model Validation
-	สีดำ = พื้นที่ที่เกิดไฟจริง (Observed Burned Area) 
-	สีน้ำเงิน = พื้นที่ที่โมเดลทำนายถูก (Correct Prediction)

ผลการตรวจสอบพบว่าโมเดลมี Accuracy เท่ากับ 68.45% ซึ่งอยู่ในระดับปานกลางถึงดี

ข้อสรุป:
-	โมเดลสามารถระบุพื้นที่เสี่ยงได้ในระดับที่น่าเชื่อถือ
-	เหมาะสำหรับการวิเคราะห์แนวโน้มเชิงพื้นที่ แต่ยังมีข้อจำกัด เช่น ความละเอียดข้อมูล และการไม่รวมปัจจัยด้านมนุษย์
## สรุปผล (Conclusion)
โมเดลนี้ใช้ปัจจัยด้านความแห้ง อุณหภูมิ เชื้อเพลิง และประวัติไฟป่า ร่วมกับวิธี Weighted Linear Combination เพื่อสร้างแผนที่ความเสี่ยงไฟป่า แม้ Accuracy จะอยู่ที่ประมาณ 68% (ระดับปานกลาง) 
แต่ยังสามารถใช้เพื่อวิเคราะห์แนวโน้มพื้นที่เสี่ยงและสนับสนุนการวางแผนจัดการไฟป่าในระดับจังหวัดได้
## ข้อจำกัด (Limitations)
-	ความละเอียดข้อมูล (MODIS 500m–1km)
-	ไม่รวมปัจจัยด้านมนุษย์ เช่น การเผา
-	ไม่มีตัวแปรสภาพอากาศ เช่น ลมและความชื้น
-	การใช้ median อาจลดความแปรผันของข้อมูล
## สิ่งที่ค้นพบ (Key Findings)
-	พื้นที่เสี่ยงสูงกระจุกตัวในบางส่วนของจังหวัด
-	Dryness เป็นปัจจัยที่มีอิทธิพลสูงที่สุด
-	โมเดลสามารถตรวจจับพื้นที่ที่เกิดไฟจริงได้ประมาณ 68%
-	โมเดลมีความเสถียรในภาพรวม แต่ยังมีบางพื้นที่ไม่แน่นอน
## แนวทางพัฒนาในอนาคต (Future Improvement)
-	เพิ่มตัวแปรด้านสภาพอากาศ เช่น ลม และความชื้น
-	เพิ่มปัจจัยด้านมนุษย์
-	ใช้ข้อมูลที่มีความละเอียดสูงขึ้น
-	ทดลองใช้ Machine Learning เปรียบเทียบกับ MCA
